# Innovative-search
Innovative Search Dashboard is a web application that allows users to search multiple platforms from one place.
It includes search options for Google, Gemini, ChatGPT, Bing, GitHub, YouTube, Wikipedia, and Amazon.
The project also provides an auto-suggestion feature that displays keywords while typing.
A special search section is included for cricket-related information and suggestions.
This project is built using HTML, CSS, and JavaScript.
